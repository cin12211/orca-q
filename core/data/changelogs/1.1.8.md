# 🎉 v1.1.8 Changelog

## ✨ New Features

- **Interactive Chart Builder**: Added a new Chart tab for raw query results, allowing users to dynamically visualize query outcomes using Line, Bar, Pie, Heatmap, Radar, and Scatter charts. Includes a sidebar configuration panel to customize x-axis, y-axis, group-by fields, row limits, sorting, and aggregate functions (Sum, Count, Average, Max).
- **Direct Result Export**: Added an Export dropdown button to the raw query result control bar, enabling direct export of query results to CSV, JSON, or TSV formats.
