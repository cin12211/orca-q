# 🎉 What's New in v1.1.2

Version 1.1.2 extends SQLite compatibility and introduces a new Redis database option.

## ✨ New Features

### SQLite

- **Managed SQLite Support:** SQLite now supports D1 and Turso deployments.

### Database Support

- **Redis Added:** Introduced Redis as a new supported database type.

## 🔧 Improvements

- **SQLite Coverage:** Improved SQLite support to better handle both local and managed SQLite workflows.
