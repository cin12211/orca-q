# 🎉 What's New in v1.0.26

We are excited to announce version 1.0.26, enhance support view virtual table (View, Materialized View) in quick query

## ✨ New Features

### Command Palette

- **Global Command Palette:** Press `Cmd+K` / `Ctrl+K` to open a searchable command palette from anywhere
- **Prefix-based Search:** Use prefixes to filter by category — `t:` tables, `v:` views, `f:` functions, `tab:` switch tabs, `erd:` open ERD, `>` run commands
- **Direct Commands:** Reload Schema, Reload App, Close App, and open Settings directly from the palette
- **Keyboard Navigation:** Full keyboard-first experience with `↑↓` navigate, `↵` select, `Esc` close

## 🔧 Improvements

### Quick Query

- **Support View Virtual Table:** Enhanced support view virtual table (View, Materialized View) in quick query

### Raw Query

- **Explain Query:** Allow user to view EXPLAIN query execution plans
- **Format SQL:** Enhanced format shortcut function
