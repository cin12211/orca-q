# 🎉 v1.1.9 Changelog

## ✨ New Features

- **Tab Keyboard Shortcuts**: Added keyboard shortcut support to easily manage tabs:
  - Press `⌘W` (or `Ctrl+W`) to close the active tab.
  - Press `⌘⌥T` (or `Ctrl+Alt+T`) to close all other tabs.
- **Tab Context Menu Polish**: Updated shortcut hints in the tab context menu, mapping "Close other" to the new `⌘⌥T` shortcut.

## 🔧 Improvements & Fixes

- **Toggle Right Sidebar Shortcut**: Updated the shortcut to toggle the right sidebar from `⌘⇧B` to `⌘⌥B` to avoid potential browser/system conflict and ensure layout consistency.
