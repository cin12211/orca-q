# 🎉 v1.1.6 Changelog

## ✨ New Features

- **Raw Query**: Supported inline data editing and relation preview directly from query results.
- **Table Grid Refactor**: Refactored the AG Grid wrapper, unified global component sizes, and updated Storybook.
- **Routine Editor**:
  - Added a Reload button and keyboard shortcuts (`⌘R` / `Ctrl+R` / `F5`) to refresh view/function definitions from the database, prompting a confirmation if there are unsaved changes.
  - Fixed the keyboard shortcut `⌘S` / `Ctrl+S` failing to save routine definitions.
  - Fixed an event parameter bug where clicking "Save to database" inside the preview dialog passed a `MouseEvent` instead of the SQL string, causing database execution errors.

## 🔧 Bug Fixes

- **Reset All Data (Settings)**:
  - Fixed the blank screen issue in the Electron app by replacing the `/` redirect (which loaded `file:///`) with an in-place reload (`location.reload()`).
  - Fixed database clear failures caused by foreign key violations on the SQLite pool by wrapping all table deletions in a single transaction in the correct dependency order.
