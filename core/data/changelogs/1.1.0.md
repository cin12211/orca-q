# 🎉 What's New in v1.1.0

We are excited to announce version 1.1.0, marking the official release of the desktop app with expanded platform and database support.

## ✨ New Features

### Desktop App

- **Official Desktop Release:** The desktop app is now officially released.
- **Cross-platform Support:** Native desktop support is now available for both macOS and Linux.

### Database Support

- **Beta Database Drivers:** Added beta support for additional database types including MySQL, OracleDB, MariaDB, and SQLite3.

## 🔧 Improvements

- **Connection Coverage:** Expanded the supported connection matrix so more database engines can be tested directly in the desktop workflow.
