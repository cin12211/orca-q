# 🎉 v1.1.10 Changelog

## ⚡ Improvements & Performance

- **Enhanced Bulk Operations**: Optimized bulk delete and bulk update performance across MySQL, Oracle, PostgreSQL, and SQLite table adapters:
  - Enabled batch query execution within transaction blocks (`BEGIN`/`COMMIT`/`ROLLBACK`) to reduce network roundtrips.
  - Properly managed connection acquisition and release in all adapter environments to avoid leaks.
  - Replaced hardcoded limits with a centralized `BULK_CHUNK_SIZE` config constant.
- **UI Enhancements**: Added a loading overlay during query updates and optimized query execution state controls in the result tab.
- **Duplicate Field Join Support**: Fixed query rendering in raw query when joining tables sharing identical field/column names.
